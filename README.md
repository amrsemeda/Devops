## Initialization
```bash
sudo apt update
sudo apt install python3.10-venv
python3 -m venv venv
source venv/bin/activate
pip install flask
pip freeze > requitments.txt
```

## Development
```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```
